Welcome to the Prime Checker API!

This API provides a way to check if a number is prime.

Usage

To check if a number is prime, use the following endpoint:

GET /is_prime?number=<integer>

Example usage: /is_prime?number=17

Responses

  • {"number": , "is_prime": true} if the number is prime
  • {"number": , "is_prime": false} if the number is composite
  • {"number": , "is_prime": null} if the result is undetermined

Important Notes

  • Primality is determined using trial division checking all prime factors up to {{ limit }}.
  • A result of "true" indicates that no prime factors were found and the number is confirmed prime.
  • A result of "false" indicates that a prime factor was found and the number is confirmed composite.
  • A result of "null" means the number could not be confirmed prime as it is above the checkable limit.
