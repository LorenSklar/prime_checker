Welcome to the Prime Checker API!

This API provides a way to check if a number is prime.

Usage

To check if a number is prime, use the following endpoint:

POST /v2/is_prime

Request body (JSON):

{
"number": 17
}

Example usage with curl:

curl -X POST http://127.0.0.1:5010/v2/is_prime \
-H "Content-Type: application/json" \
-d '{"number": 17}'

Responses

  • {"number": , "is_prime": true} if the number is prime
  • {"number": , "is_prime": false} if the number is composite

Important Notes

  • Primality is determined using the SymPy library.
  • For numbers greater than 2^64, a probablistic prime check is used.
  • For more information see the Sympy documentation
